Do you want to contribute with this project?

Be part of this!

We would be so happy if you contribute to improve Graviton. There are many different ways to help us with. The only way you can help (by now) is by creating pulls requests on the project Repository.

Things which you can contribute on:

  • Translating in other languages or improve one.
  • Request a new feature.
  • Fix or report bugs and glitches.
  • Improve some script or UI/UX design.
  • Create your own themes and plugins.

Buy a coffee!

You can support graviton by donating some money via PayPal, in this way we can continue developing and dedicating time to great quality free software.

By donating you can access to:

  • Get priority support.
  • Be recognized in the community.
  • Featured positioning of your packages.
  • Help Open-Source developers.
  • And many other benefits.
